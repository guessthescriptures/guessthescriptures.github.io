How to Play

Introduction

Guess the Scriptures is a game in the style of a trivia game with the goal of strengthening one's memory of the Scriptures that teach certain truths.

How to Play Guess the Scriptures

To play Guess the Scriptures, select New Game from the main menu.

The game consists of a number of randomly selected questions. (By default, it is only one question, but the available questions and the number of questions per game can be adjusted in Settings.) The game presents one question at a time. Each question must be answered before the game can present the next question. The questions are based from Appendix A Truths We Love to Teach from the brochure Love People—Make Disciples published by Jehovah's Witnesses.

Below the presented question is a 4 x 2 grid (or 2 x 4 grid for smaller devices) of at most 8 boxes each with an unchecked checkbox and a scripture associated with it. Checking a checkbox is a selection of the associated scripture. Check the checkboxes that answer the presented question.

You can select the text of the scripture to read the full text in a pop-up dialog box before or after the question is answered. The full texts of the scriptures are from the modern-language New World Translation of the Holy Scriptures. To close the pop-up dialog box, select the close button in the upper right corner of the pop-up dialog box.

After selecting the scriptures that answer the presented question, to submit the selected scriptures, select the button labeled Submit Selected Scriptures. A pop-up dialog box will display the results. To close the pop-up dialog box, select the close button in the upper right corner of the pop-up dialog box. The boxes that each contain the correct scripture are highlighted in green. A point is given to your score if you selected only and all the correct scriptures for the presented question.

To proceed to the next question, select the button labeled Next Question.

To restart the game at any time, select the button labeled Restart Game.

To quit the game and return to the main menu, select the link labeled Return to Main Menu.

The goal of the game is to answer all the questions correctly.

Review the Questions and Their Correct Scriptures

Before or after playing a game, you can review the questions that may be presented in the game along with their correct scriptures.

To review the questions and their correct scriptures, select Review from the main menu.

One question is presented at a time.

To select a question to be presented, use the selection box labeled Question and select the question you want to have presented.

Below the presented question is a 4 x 2 grid (or 2 x 4 grid for smaller devices) of at most 4 boxes highlighted in green each with a scripture.

You can select the text of the scripture to read the full text in a pop-up dialog box. To close the pop-up dialog box, select the close button in the upper right corner of the pop-up dialog box.

To present the previous question and its correct scriptures, select the button labeled Previous Question.

To present the next question and its correct scriptures, select the button labeled Next Question.

To return to the main menu, select the link labeled Return to Main Menu.

Adjusting the Settings

You can adjust the settings for the game such as which questions can be available for the game and for review and how many questions per game.

To adjust the settings for the game, select Settings from the main menu.

Settings

The following discusses each setting.

Available Questions

A column of boxes is presented each with a checkbox and a question associated with it. A checked checkbox indicates the associated question is available for the game and for review. An unchecked checkbox indicates otherwise. You can check which question to be available. However, at least one checkbox can remain checked.

Number of Questions Per Game

A selection box is presented with options each containing a number of the questions from the available questions to be selected for the game and for review. The number of options increase or decrease based on the number of available questions. You can select how many questions from the available questions to be selected for the game and for review.

Buttons

The following discusses each button.

Apply Current Settings

To apply the current settings to take effect in this window or tab, select the button labeled Apply Current Settings. The applied current settings will remain in effect until this window or tab is refreshed or closed. To have the applied current settings persist after this window or tab is refreshed or closed, you will need to save the current settings to local storage.

Save Current Settings

To save the current settings to local storage, select the button labeled Save Current Settings. The saved current settings will take effect in a new or refreshed window or tab.

Delete Saved Settings

To delete any saved settings in local storage, select the button labeled Delete Saved Settings. If the current settings are not saved to local storage, then under a new or refreshed window or tab, the default settings are used.

Reset to Default Settings

To reset the current settings to the default settings, select the button labeled Reset to Default Settings. You will need to apply and/or save the current settings in order for the default settings to take effect.

To return to the main menu, select the link labeled Return to Main Menu.
